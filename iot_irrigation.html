Overview

This IoT-enabled system automates irrigation using soil moisture sensors and allows remote control and monitoring via a web interface or mobile app also via Amazone Alexa and Google Assistant.

Components

  • ESP8266 Wi-Fi Module
  • A Soil Moisture Sensor
  • A Water Pump
  • A Two Channel Relay Module
  • A Light Bulb

How It Works

Sensors measure soil moisture levels, and the system send the sensor data to cloud. Users can monitor and control the system remotely via app and google home.
